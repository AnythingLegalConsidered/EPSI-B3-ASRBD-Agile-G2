# Definition of Ready (DoR) & Definition of Done (DoD)

## Definition of Ready (DoR)

Une User Story est prête quand :

- [ ] Rédigée au format "En tant que... je veux... afin de..."
- [ ] Critères d'acceptation définis et testables
- [ ] Estimation en points (Fibonacci) faite via Planning Poker
- [ ] Compréhensible par toute l'équipe
- [ ] Dépendances identifiées

## Definition of Done (DoD)

Une User Story est terminée quand :

- [ ] Fonctionnelle et testée
- [ ] Déployée sur Kubernetes
- [ ] Documentée
- [ ] Code/configuration poussé sur le repo
- [ ] Issue GitHub fermée avec commentaire de résultat
